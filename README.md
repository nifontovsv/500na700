# 500na700

Сайт креативного агентства **500 на 700**: лента новостей, отдельные страницы статей и форма обратной связи в модальном окне.

**[Посмотреть сайт →](https://nifontovsv.github.io/500na700/)**

## Описание

- **Главная** — список новостей с картинками, датой и кратким описанием; клик по карточке ведёт на страницу статьи.
- **Страница новости** — статья с изображением, заголовком, датой и полным текстом; внизу футер с логотипом.
- **Header** — логотип (ссылка на главную), кнопка «Связаться с нами» открывает модальное окно с формой.
- **Форма** — имя, телефон (с маской), email, согласие на обработку данных; валидация и сброс ошибок при вводе.

Данные новостей подгружаются из `public/api/news.json`. Сайт собирается в статический экспорт и деплоится на GitHub Pages.

## Стек

- Next.js 16 (App Router)
- React 19
- TypeScript
- SCSS Modules
- react-imask (маска телефона в форме)

## Разработка

```bash
npm install
npm run dev
```

## Структура

- `app/` — страницы (главная, `/news/[id]`)
- `components/` — Header, NewsList, Form, Modal
- `public/api/news.json` — данные новостей
- `.github/workflows/deploy.yml` — деплой на GitHub Pages
